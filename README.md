MEDISENSE-AI 🏥🤖

AI-powered healthcare assessment platform built with React, Node.js, Express, and MySQL. The system provides symptom-based health analysis, risk prediction, patient monitoring, and secure authentication while demonstrating advanced DBMS concepts in a real-world healthcare application.

✨ Features
🤖 AI-assisted symptom assessment
📊 Automated risk prediction & monitoring
🏥 Disease matching using MySQL stored procedures
🔐 Secure JWT authentication
⚡ Real-time patient monitoring
📱 Responsive modern UI
🗄 Advanced DBMS concepts (Triggers, Views, Procedures)

⚙️ Installation
Clone Repository
git clone <repository-url>
cd MEDISENSE-AI
Install Dependencies
# Frontend
cd frontend
npm install

# Backend
cd ../backend
npm install
Run Application
# Frontend
npm run dev

# Backend
npm start
🔑 Environment Variables

Create a .env file inside the backend directory:

PORT=5000
MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=yourpassword
MYSQL_DATABASE=medisense_ai
JWT_SECRET=your_secret_key
